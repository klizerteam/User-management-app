# FastAPI Users API

# Setup Instructions
1. Clone the repository
2. Create virtual environment
3. Install dependencies
4. Run the server

# API Endpoints
POST   /users
GET    /users
GET    /users/{id}
PUT    /users/{id}
DELETE /users/{id}

# Validation
- Backend validation using Pydantic
- Email and role validation enforced

# Tech Stack
- FastAPI
- Pydantic
- Uvicorn
